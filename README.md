# Projection HTML5

Projection for church

Technology Stack
----------------

1. Bootstrap
2. Jstree
3. Reveal.js
4. jQuery
5. FontAwesome

Runnning on browser
-------------------

* Mozilla Firefox (it works)
* Google Chrome ( work with limitation: "it need import manually data.json" )

Usage
-----

1. Run index.html
2. Allow popup
3. If you are not using **Mozilla Firefox** you 'll need open manually data/data.json
